# Setup Tools

프로젝트 초기화 및 설정 도구들입니다.

## 스크립트 설명

### Setup-Overview-Resources.ps1
- **목적**: Overview 콘텐츠 리소스 설정 (PowerShell)

### setup-overview.ps1
- **목적**: Overview 폴더 구조 생성 및 리소스 복사

### setup_overview.bat
- **목적**: Batch 버전의 Overview 셋업

### setup_overview.py
- **목적**: Python 버전의 셋업 (크로스 플랫폼)

## 버전 선택 가이드

| 환경 | 권장 버전 |
|------|---------|
| Windows PowerShell | setup-overview.ps1 |
| Windows CMD | setup_overview.bat |
| 크로스 플랫폼 | setup_overview.py |
