# Profile Tools

PowerShell 프로필 설정 및 관리 도구들입니다.

## 스크립트 설명

### PowerShell_Profile_Fixed.ps1
- **목적**: PowerShell 프로필 자동 설정 (수정된 버전)
- **결과**: 편의 함수 추가 (cdpoly, pathguide, 등)

### PROFILE_CLEAN.ps1
- **목적**: PowerShell 프로필 정리 및 초기화

## 편의 함수 (Profile_Fixed 적용 후)
- cdpoly - 프로젝트 폴더로 즉시 이동
- pathguide - 경로 처리 가이드 열기
- contentguide - 콘텐츠 구조 가이드 열기
- gentsguide - MarsMaker AI 가이드 열기
